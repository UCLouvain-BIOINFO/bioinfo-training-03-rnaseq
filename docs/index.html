RNA-Seq analysis with R and Bioconductor

RNA-Seq analysis with R and Bioconductor

Laurent Gatto and Axelle Loriot

2026-05-18

Preamble

RNA sequencing (RNA-Seq) is a powerful technology that is now commonly used to measure gene expression. A differential expression analysis is a statistical analysis that aims to identify quantitative changes in gene expression levels between different experimental groups. The course is aimed at biologists who are already familiar with the R language and who want to learn how to do a differential expression analysis on their RNA-sseq data.

The course will provide:

  • Overview on RNA-seq data processing, exploring the different steps from raw reads to the generation of a gene count matrix.
  • Introduction to theoretical concepts behind a differential expression analysis: linear models, multiple testing correction, principal component analysis, negative binomial distribution, and gene set enrichment analysis.
  • Numerous hands-on exercices.
  • A complete real-life RNA-Seq DE analysis running the DESeq2 package step by step.

Prerequisits

Participants should be familiar with the basix syntax of R and ideally also the tidyverse. They must bring a laptop with a Mac, Linux, or Windows operating system (not a tablet, Chromebook, etc.) that they have administrative privileges on. They should have installed R (https://cloud.r-project.org/) and Rstudio (https://rstudio.com/products/rstudio/download/#download).

Installation instructions

Bioconductor[^about-bioc] packages can be installed with the BiocManager::install() function from the BiocManager package, that can be installed from CRAN with the usual install.packages() function.

[^about-bioc] We will explain more about the Bioconductor projet later, when learning about the SummarizedExperiment class.

pks <- c("tidyverse", "patchwork", "remotes",
         "SummarizedExperiment", "DESeq2",
         "AnnotationDbi", "factoextra", "pRolocdata",
         "org.Hs.eg.db", "GO.db", "msigdbr", "apeglm",
         "clusterProfiler", "airway")

if (!require("BiocManager"))
    install.packages("BiocManager")

BiocManager::install(pks)
